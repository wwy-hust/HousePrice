## 2026-06-28 新增社零数据可视化模块

### 新增功能：社会消费品零售总额数据可视化
- 基于房价系统架构，新增社零数据采集和图表展示
- 数据源：国家统计局 / AKShare (`macro_china_consumer_goods_retail`)

### 新增文件
- `retail_data_collector.py` - 社零数据采集脚本
  - 通过 AKShare 获取社零总额历史数据（2020年至今）
  - 手动录入限额以上分品类数据（家电、金银珠宝、汽车等16个品类）
  - 生成 `results/retail_data.json`
- `web/retail.html` - 社零数据可视化页面
  - 统计卡片：社零总额、商品零售YoY等
  - 折线图：社零总额月度趋势 + 同比增速柱状图
  - 分品类表格：按必选消费/可选消费/居住相关/交通能源分组

### 修改文件
- `web_server.py` - 新增 `retail_data.json` 到允许文件列表
- `web/index.html` - 首页添加"社零数据"导航入口

### 访问信息
- 房价数据: http://localhost:8000
- 社零数据: http://localhost:8000/retail.html
- 数据范围: 2020年至今，每月更新
